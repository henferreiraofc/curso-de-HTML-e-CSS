Anotações sobre Simbolos e Emojis

ADD Simbolos especiais:

Colocando "& reg;", tudo junto, forma o simbolo de marca registrada ®.

"& copy", tudo junto, é igual a ©.

"& trade", tudo junto, é igual a ™.

OBS:

1 - No PDF da aula tem mais símbolos disponiveis para usar caso necessario.

2 - Alguns símbolos, para aumentar o tamanho, iniciam com a primeira letra do comando "Maiúscula" e para o simbolo ser pequeno tem a letra "Minúscula".

ADD Emojis:

1 - Podemos add emojis usando codigos prontos nas bibliotecas disponiveis, más alguns navegadores podem não aceitar.

Outra forma é usando o comando:
"&#x"+"codigo do emoji;".
Exemplo "&#x"+"1F596;" é igual a 🖖.
Todo emoji começa com "&#x", assim o navegador entende que se trata de um emoji.
