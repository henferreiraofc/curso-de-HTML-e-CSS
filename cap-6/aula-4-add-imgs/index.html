Adicionando Imagens

Para add imagens, usamos a TAG "img", com ela teremos o "src" para add a imagem e o "alt" para deixar a descrição da imagem caso não carregue ou tenha um deficiente visual venndo a pagina.

Add imagens externas

Podemos add imagens usando diretamente da internet, sem a necessidade de baixarmos elas.
Para isso nós copiamos a URL da imagem e colamos no local "src".
