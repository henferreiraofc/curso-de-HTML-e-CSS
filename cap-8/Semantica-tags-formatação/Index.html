Semantica e TAG's de formatação

O que é a Semantica no HTML5?

Semântica é o que da significado para o HTML5.

Para o HTML5, usar as TAG's é importante para dar significado as formas que estão sendo usadas no conteúdo. Ao invés de ser apenas formatações visuais, o uso das TAG's mostra para os mecanismos de busca que ali significa algo.

TAG's Semânticas

< adress >< /adress > é a TAG usada para por uma localização que pode ser aberta no Google Maps.
Exemplo:
R. Eng. João Ortiz Monteiro - Morumbi, São Paulo - SP

Negrito:

TAG = < strong > < /strong >
Exemplo: Olá, Mundo

Italico:

TAG = < em > < /em >
Exemplo: estou escrevendo em italico.

Marca Texto:

TAG = < mark > < /mark >
Exemplo: Queria deixar esse texto destacado.
OBS: Para mudar a cor teria que ser com CSS.

Traço na Palavra (texto visivel mas excluido):

TAG = < del > < /del >
Exemplo: Essa palavra foi desconciderada nessa frase, voce pode le-la, mas não leve em consideração.

Traço em Baixo da Palavra:

TAG = < ins > < /ins >
Exemplo: Gostaria de destacar a importancia de usar essa TAG destacando as palavras.

Envelopamento de Abreviaturas (TAG's):

TAG = abb na barra de pesquisa. Para abrir a aba de pesquisa, usamos o comando CTRL + SHIFT + P.

Texto Sobrescrito(Cima) e Subscrito(Baixo):

TAG Sobrescrito = < sup > < /sup >
Exemplo: X4
TAG Subscrito = < sub > < /sub >
Exemplo: H2O

Trechos de Código:

TAG = < code > < /code >
Exemplo: num = int(input('Digite um número')) if num % 2 == 0: print(f'O número é PAR') else: print(f'O número é IMPAR')
OBS: Dessa forma as letras não irão pular linhas e ficarão mono-espaçadas, para que fique igual ao que foi escrito, usamos a TAG < pre > < /pre >
Exemplo com a TAG pre: 

                num = int(input('Digite um número'))
                if num % 2 == 0:
                    print(f'O número é PAR')
                else:
                    print(f'O número é IMPAR')

OBS2: A TAG pre pode ser usada para deixar linhas igual foram escritas na hora de programar.

Citações:

Temos dois tipos de Citações, temos as aspas" e temos outro tipo de citação que forma um bloco com espaçamento para o lado.

Aspas:

TAG = < q > < /q >
Exemplo: Estou cursando o Curso em Video.

Espaçamento:

TAG = < blockquote> < /blockquote >
Exemplo:
todo esse conteudo esta com um espaçamento, igual a quase todos os parágrafos deste codigo.

Texto Invertido:

TAG = < bdo dir="" >< /bdo >
Exemplo: esse texto esta invertido.
OBS: A TAG possui um PARAMETRO dir = " " que precisa ser preenchido com:
rtl = da direita para esquerda(invertido).
ltr = da esquerda para direita(normal).

Letras Grandes e Pequenas:

No HTML5 a TAG para deixar o as letras grandes ficou obsoleta, mas para usar é a TAG < big >.
A TAG para deixar pequena não esta obsoleta, talvez por que ainda seja usada para fazer observações em alguma parte do site.

Pequena:

TAG = < small >< /small >
